## 강의 고유 하드 게이트 2개 (글엔 없음)

- **⏱ 시간 예산 게이트** — 모듈 분(分) 합 = 총 시간(±버퍼). **산술이라 거짓 못 함.** 초과면 통과 불가 → **비목표(1b)로 쳐내** 줄인다. "다 넣고 싶다"를 막는 물리적 제약. *3과제 균등 안배* 같은 흔한 함정을 한 줄 약속에 맞춰 집중으로 재편하게 만드는 결정적 강제력.
- **🪧 1슬라이드=1메시지 게이트** — 한 슬라이드 메시지 2개↑면 FAIL → 쪼갠다(*말할 것 vs 띄울 것* 분리). *읽는 슬라이드 = 죽은 슬라이드.*

## §검토 체크리스트 (분리된 검토자에게 그대로 전달)

**(가) 내용 정합 + 강의 고유**
- ☐ 한 줄 약속이 *모든 모듈*을 꿰는 중심인가
- ☐ **비목표 준수** — 쳐낸 곁가지가 돌아오지 않았나
- ☐ 모듈별 takeaway가 *증거*로 정의됐나(모호어 아님)
- ☐ 섹션 × {사례,근거} 쌍 맞물렸나
- ☐ **⏱ 시간 예산** 맞나(모듈 합 = 총 시간) / **🪧 1슬라이드=1메시지** 지켜졌나
- ☐ 청중 *졸 구간*(상호작용 공백 30분+) 없나
- ☐ *말할 것 vs 띄울 것* 분리됐나(슬라이드가 대본 아님)

**(나) 라벨 vs 구조역전**
- ☐ 한 줄 약속이 자료 라벨을 그대로 베꼈나, 새 각도인가 → 라벨이면 관문 재오픈

**(다) 공적 표현·DNA**
- ☐ P1 훈계조 / P2 자신감 비대칭 / P13 은유 / P15 구체

**(라) 레퍼런스 대조 (잘 된 강의 PPT가 있을 때만)**
- ☐ 구성·전달·슬라이드 양식 강점 살렸나 / 의도적 이탈 정당한가 / 그대로 베꼈나
