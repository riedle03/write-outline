- **DNA: 사례(3)·근거(4)는 쌍.** 섹션마다 맞물려야 둘 다 🟢.
- **근거 유형 게이트(4):** 개요 전 "수치형 vs 서사형" 판정. 근거 약하면 "채워라" + "(수치)프레임 버리고 장면·인용으로" 동등 후보. 레퍼런스(서사형 기사)가 유형을 잡아준다.
- **구조 정렬(5):** 섹션 순서 = 강조점 랭킹이 아니라 **사례의 시간·인과 사슬**.
- 레퍼런스는 형태 칸(5·6·톤)만. 판단 칸(1·1b·1c·2·3)엔 절대 안 닿는다.

## 도전 항목 (판단형 칸, P1~P16·공적 표현)

| 칸 | 도전 항목 |
|----|------|
| 핵심 메시지 | "자료가 이미 단 라벨을 베낀 것 아닌가?"(관문 ⓑ) · "통념 세워 꺾는 설교 틀인가?"(P1) |
| 둘째 논지 | "주 논지만 있고 떠받치는/반박하는 또 하나의 논지가 비지 않았나?" |
| 독자·목적 | "학생·집단을 단정·일반화('모두…')하지 않았나?"(P2) · "구조적 어려움을 미화하지 않았나?"(P3) |
| 중심 사례 | "추상이 아니라 실제 이름·맥락이 들어갔나?"(P15) · "은유로 때운 장면 아닌가?"(P13) |

> 문장 차원(P9 의인화·P10 시제·P11 한자투·P14 압축·P16 영문병기)은 개요가 아니라 다운스트림(humanize-korean) 몫. 도전은 *개요에 박히는* 구조·진단 차원만 친다.

## §검토 체크리스트 (분리된 검토자에게 그대로 전달)

**(가) 내용 정합**
- ☐ 의뢰서 요소를 개요가 모두 덮는가
- ☐ 강조점이 모든 섹션을 꿰는 **중심**인가, 한 단락 장식인가
- ☐ **둘째 논지 자리 있나** — 둘째 논지가 섹션으로 들어왔나
- ☐ **비목표 준수** — 버린 곁가지가 슬그머니 돌아오지 않았나
- ☐ **섹션 × {사례,근거} 매트릭스** — 각 주장 섹션이 사례·근거를 쌍으로 가졌나
- ☐ **근거 유형 정합** — 수치형/서사형 판정과 실제 근거 열이 맞나(서사형인데 효과크기 깔지 않았나)
- ☐ **섹션 순서 = 사례 인과/시간 순**인가(강조점 랭킹으로 뒤집지 않았나)
- ☐ 구조가 도입→전개→마무리로 *진행*하나(나열 아님)

**(나) 라벨 vs 구조역전**
- ☐ 강조점·소제목이 자료가 이미 단 라벨을 그대로 베꼈나, 자료 구조를 뒤집은 새 각도인가
- → 라벨 그대로면 FAIL → 핵심 메시지 관문(§3) 재오픈

**(다) 공적 표현·DNA (개요 차원)**
- ☐ P1 훈계조 대비 구조 / P2 자신감 비대칭 / P3 어려움 솔직 인정 / P8·P12 메타·설계해설 / P13 은유 / P15 구체 / P6 인접 중복

**(라) 레퍼런스 대조 (있을 때만)**
- ☐ 레퍼런스의 구성·형식·문체 강점을 살렸나 / 의도적 이탈이 정당한가 / 그대로 베껴 복제가 됐나
